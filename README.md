#twerk.js
shake dat ass

**Demo**: http://codevinsky.github.io/twerk.js

###Requires jQuery

1. Include the two twerk files after jquery.

2. Assign the "twerk" class to any element you want to twerk.

	``` html
		<div class="twerk">
			<h1>Let's twerk</h1>
		</div>
	```

3. Call *twerk.getLow()*

	``` html
		<button type="button" onclick="twerk.getLow()"> 
			Get low
		</button
	``` 

4. Party.
